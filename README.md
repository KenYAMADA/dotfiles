# dotfiles

## what is this

環境設定の自動化を目的とする。
macOSとLinuxを対象としている。

## how to use

### Linux and macOS
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/kenyamada/dotfiles/main/install.sh)"
```

### Windows (BETA)
'''Power Shell
iex "& {$(irm 'https://raw.githubusercontent.com/kenyamada/dotfiles/main/install.ps1')}"
```
